## Docker
常用指令
>1.\$sudo docker ps # 列表显示容器
2.\$sudo docker images # 列表显示镜像
3.\$sudo docker run -it xxx:yyy /bin/bash # 创建xxx镜像:yyytag的实例
4.\$sudo docker container exec -it \[containerID]]  /bin/bash #进入对应ID的容器
